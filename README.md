## 익명 투표 모바일 앱 데이터 분석과 번역 및 질문 추천 모델 개발

### 청소년대상 익명투표 어플리케이션 서비스 이용 패턴 분석

- 사용자 최대 가입 후 3개월차에 리텐션 비율이 10%대로 급감하는 문제 발생
- 사용자 이탈의 주요 원인 중 "질문이 재미없어서"를 해결하기 위한 방안 제시
- 기존 질문 데이터를 분석하여 핵심 단어 추출 및 transformer 기반 분류 모델 구성
- 추출된 핵심 단어를 기반으로 Gemini api에 입력하여, 사용자 맞춤형 질문 추천 시스템 구축
- 흥미를 유발하는 질문을 통해 약 11만명의 이탈 방지를 추정
